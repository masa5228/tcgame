# 画像ファイル配置ガイド

このディレクトリに以下の画像を配置してください：

## 必須画像

### 1. logo.png
- **用途**: サイトロゴ
- **推奨サイズ**: 200x60px (横長)
- **フォーマット**: PNG (透過背景推奨)
- **説明**: ヘッダーに表示されるゲームのロゴ

### 2. hero-forest.png
- **用途**: メインビジュアル
- **推奨サイズ**: 1920x800px (16:9比率推奨)
- **フォーマット**: JPG または PNG
- **説明**: トップページのヒーローセクションに表示される背景画像
- **内容**: 朝靄の森、スカンジナビア風の山と森林風景

### 3. intro-scene.png
- **用途**: 世界観紹介セクション
- **推奨サイズ**: 600x600px (正方形、または4:3)
- **フォーマット**: JPG または PNG
- **説明**: 世界観紹介セクションの左側に表示される画像
- **内容**: 森の村、木製アイテム、動物キャラクターなど

## オプション画像

### 4. texture-wood.png
- **用途**: 背景テクスチャ（オプション）
- **推奨サイズ**: 1920x1080px (タイル可能なパターン)
- **フォーマット**: PNG
- **説明**: セクション背景に使用可能な木のテクスチャ

## 注意事項

- すべての画像はWeb最適化されたフォーマット（圧縮済み）を使用してください
- 高解像度ディスプレイ対応のため、推奨サイズの2倍の解像度で作成することを推奨します
- ファイル名は必ず上記の通りにしてください（大文字小文字を含む）
- パスを変更する場合は `src/config/images.js` を編集してください

## プレースホルダー画像

開発中にプレースホルダー画像を使用する場合は、以下のサービスが便利です：
- https://placehold.co/
- https://picsum.photos/

例：
- logo: https://placehold.co/200x60/5a7c5a/white?text=T!C!Game
- hero: https://picsum.photos/1920/800?nature
- intro: https://picsum.photos/600/600?forest
